    API Reference crux_interaction v0.1.0-dev

    Modules

    Crux.Interaction.ApplicationCommand

    Behavior providing __crux_command__/0 to access underlying command data.

    Crux.Interaction.ApplicationCommand.ChatInput

    Provides a DSL to define chat input application commands A.K.A. slash commands.

    Crux.Interaction.ApplicationCommand.Exceptions.InvalidChoice

    Raised when an invalid application command chat input choice name or description was specified.
    Also raised when too many or duplicated choices were specified.
    See valid_name?/1 and valid_choice?/2

    Crux.Interaction.ApplicationCommand.Exceptions.InvalidDefaultPermission

    Raised when an invalid @default_permission was specified for an application command chat input.
    I.e. not a boolean.
    See: valid?/1

    Crux.Interaction.ApplicationCommand.Exceptions.InvalidDescription

    Raised when an invalid description was specified anywhere.
    See: valid?/1

    Crux.Interaction.ApplicationCommand.Exceptions.InvalidName

    Raised when an invalid name was specified anywhere.
    See: valid?/1

    Crux.Interaction.ApplicationCommand.Exceptions.InvalidState

    Raised when an option is specified where it's not valid.
    E.g. when a subcommand/4 is specified after a string/2,3 or a choice/2 in a subcommand/4's do block.
    Also raised when an invalid option modifier was specified.
    E.g. when @required is used where it's not allowed.
    Also raised when too many or duplicated options were specified.

    Crux.Interaction.ApplicationCommand.Message

    Provides a DSL to define message application commands A.K.A. context menus.

    Crux.Interaction.ApplicationCommand.User

    Provides a DSL to define user application commands A.K.A. context menus.

    Crux.Interaction.MessageComponent

    Message components are interactive elements that may be added to messages or be sent as part of a response to an interaction.

    Crux.Interaction.MessageComponent.ActionRow

    An action row is a non-interactive container component for other types of components.

    Crux.Interaction.MessageComponent.Button

    Buttons are interactive components.
